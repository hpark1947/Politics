# Politics

정치, 역사, 국가, 국제정치, 사회문제에 관한 조사·분석·요약 프로젝트

## 분야

- 정치 상황 분석 및 예측
- 학자 견해 및 언론 보도 조사·분석
- 정치·사회·역사 관련 서적 분석·요약

## 구조

- `guide.md` — 프로젝트 가이드라인 (서술 원칙, 에이전트 활용 체계)
- `Prompts/` — 분석 결과 문서

## 서술 원칙

1. 전문가 수준이되 초보자도 이해할 수 있도록 자세하게 서술
2. 중요한 부분은 구체적 예시를 적시하며 설명
3. 비교·정리에 표(Table)를 적극 활용
